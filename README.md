# Simple CLI Calculator

This project is a **command-line calculator** written in Python.  
It supports **Addition, Subtraction, Multiplication, and Division** using a menu-driven interface.

---

## ## Features
- Menu-driven CLI interface  
- Supports +, -, *, /  
- Validates wrong choices  
- Handles division by zero  
- Loops until the user exits  
- Beginner-friendly and clean code  

---

## How It Works
1. User selects an operation from the menu  
2. Enters two numbers  
3. Program calls the appropriate function  
4. Displays the result  
5. Continues until the user selects exit  

---

## Code Structure

### **Functions:**
- `add(a, b)` – Returns a + b  
- `subtract(a, b)` – Returns a - b  
- `multiply(a, b)` – Returns a * b  
- `divide(a, b)` – Returns a / b, checks division by zero  

### **Main Function**
- `calculator()`  
  - Shows menu  
  - Takes user input  
  - Calls operator functions  
  - Loops until exit  

---

### **1. Save the script**
